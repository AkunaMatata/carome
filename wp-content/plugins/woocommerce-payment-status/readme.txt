== Description ==

WooCommerce Payment Status Allows users to set a status to the payment of an order; paid or not paid. Also intorduces a new column to the Orders page to indicate this status.

== Installation ==

1. Upload the entire 'woocommerce-payment-status' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
